## set-setting

Applies an option

Example:

```
> set-setting name=user.command-line-autocompletion value=False
```

## zoom-map-scale

Applies map scale. The program works in ppi (or inches to inches). So the `1` in `1:X` is implicit.

Example:

```
> zoom-map-scale 63360
```
